STEP 1: CREATE .env.local FILE
================================================================================

   create a new file named:   .env.local

2. The file should be in the same location as package.json

================================================================================
STEP 2: GET YOUR API KEYS
================================================================================

A. RAZORPAY KEYS (Payment Processing)
1. Go to: https://dashboard.razorpay.com/
2. Login to your Razorpay account
3. In the top-left, make sure "Test Mode" is selected
4. Go to Settings ‚Üí API Keys (or click "Generate Test API Keys")
5. You'll see:
   - Key Id: rzp_test_XXXXXXXXXXXX
   - Key Secret: Click "Show" or "Generate" to reveal

B. RESEND API KEY (Email Notifications)
1. Go to: https://resend.com/
2. Sign up or login to your account
3. Go to API Keys section
4. Create a new API key
5. Copy the key (starts with "re_")
6. Verify your domain (ccge.in) in the Domains section

================================================================================
STEP 3: ADD KEYS TO .env.local FILE
================================================================================

Open the .env.local file you created and paste:

NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_test_YOUR_KEY_ID_HERE
RAZORPAY_KEY_SECRET=YOUR_KEY_SECRET_HERE
RESEND_API_KEY=re_YOUR_RESEND_API_KEY_HERE

Replace YOUR_KEY_ID_HERE, YOUR_KEY_SECRET_HERE, and YOUR_RESEND_API_KEY_HERE with your actual keys.

Example:
NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_test_1234567890ABCD
RAZORPAY_KEY_SECRET=abcdefghijklmnopqrstuvwx123456
RESEND_API_KEY=re_AbCdEfGhIjKlMnOpQrStUvWx123456

SWITCHING BETWEEN TEST & LIVE MODE
================================================================================

‚úÖ **SAFE TO SWITCH ANYTIME** - No issues switching between modes!

**To switch from TEST to LIVE mode:**
1. In Razorpay Dashboard, toggle to "Live Mode"
2. Generate Live API Keys (rzp_live_...)
3. Update .env.local with LIVE keys:
   ```
   NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_live_YOUR_LIVE_KEY
   RAZORPAY_KEY_SECRET=YOUR_LIVE_SECRET
   ```
4. Restart server: npm run dev

**To switch back to TEST mode:**
1. In Razorpay Dashboard, toggle to "Test Mode"
2. Use your existing test keys
3. Update .env.local with TEST keys:
   ```
   NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_test_YOUR_TEST_KEY
   RAZORPAY_KEY_SECRET=YOUR_TEST_SECRET
   ```
4. Restart server: npm run dev

‚ö†Ô∏è  **Important:** Only use LIVE mode when ready to accept real payments!
